# SocialBu Test RSS

Use the raw `feed.xml` URL from this repo as a test RSS feed in SocialBu automation.

To trigger a new item:
1. Add a new `<item>` near the top of `feed.xml`
2. Change `guid`, `link`, and `pubDate`
3. Commit the change

Raw URL:
`https://raw.githubusercontent.com/heyumarkhan/socialbu-test-rss/main/feed.xml`
